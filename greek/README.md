# Grammatical comments

## Contract verbs
* 3 τελέω: τελέ-ει --> τελεῖ
* 17 εὐτυχέω: εὐτυχέ-ουσα -> εὐτυχοῦσα
* 33 νεῖσθαι	νέομαι

* 3 ζάω: ζά-ουσαιν --> ζώσαιν (dual dative: "for us two living")
* 30 	εἰσοράω: εἰσορά-ουσι --> εἰσορῶσι 
* 42 	δράω: δρά-ει --> δρᾷ

* 9-10: λανθάνει ... κακά
